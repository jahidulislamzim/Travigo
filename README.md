# Travigo Modern Travel Landing Page

**Travigo** is a static single-page travel landing page designed to present destinations, outdoor experiences, and structured pricing plans. The project focuses on a **minimalist, modern UI** with smooth scrolling and mobile-first responsiveness.

The main goals of the project include:

*   Building a responsive, mobile-first landing page
*   Practicing React component-based architecture
*   Implementing clean layout design using Flexbox and Grid
*   Maintaining consistent spacing, typography, and minimal UI design

## Features

*   **Hero Section:** Bold introduction with a clear call-to-action
*   **Destinations:** Curated travel cards in a structured grid layout
*   **Travel Highlights:** Regional travel inspiration with estimated drive times
*   **Outdoors Section:** Adventure-focused section highlighting nature
*   **Pricing Section:** Comparison between Standard and Premium plans
*   **Footer:** Clean navigation and contact section
*   Fully responsive, mobile-first, smooth scrolling, and minimal UI

## Installation & Setup

```

# Clone the repository
git clone https://github.com/jahidulislamzim/travigo.git

# Navigate to project folder
cd travigo

# Install dependencies
npm install

# Start development server
npm start
    
```

The app will run at: `http://localhost:3000`
